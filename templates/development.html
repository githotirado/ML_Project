Application Development

The Machine Learning app we developed is basically the 91-question questionnaire whose responses are sent as validation input data to our Random Forest Classifier trained model; the model predicts whether the questionnaire taker is an Introvert, an Ambivert or an Extrovert.

Tools and Services: The following tools were utilized in the development of the application:

  • AWS RDS: Deployed a PostgreSQL database for storage of the questions, responses and participant demographics.
  • Python: Underlying code for integrating all relevant libraries, models and data.
  • Pandas: DataFrames for preprocessing the input data and for training/testing/scoring our ML model.
  • Jupyter Notebook: Coding Pandas DataFrames, debugging.
  • Scikit Learn: Library with supervised and unsupervised learning algorithms to train/test/score/predict our model.
  • Matplotlib: Plotting our Machine Learning charts.
  • HTML/CSS: Visualization and documentation pages for our web site.
  • JavaScript: Dynamically create questionnaire HTML page from database tables.
  • SQLAlchemy: Connection from JavaScript to PostgreSQL database.
  • D3: Read questions and response data from database, add HTML to questionnaire.
  • Flask: Manage all web pages, database queries, unpickle our model, feed user data into model for validation.
  • Tableau: Visualizations dashboard of participant demographics.
  • Heroku: Deployment of application to standalone web site.

Data Source

    Kaggle.com

The questionnaire contains 91 questions and 3 demographic identifiers. The questions were presented one at a time in a random order. For each question 3 values were recorded:

A - The user's selected response. 1=Disagree, 2=Slightly disagree, 3=Neutral, 4=Slightly agree, 5=Agree I - The position of the question in the survey. E - The time elapsed on that question in milliseconds.

Application Deployment

The web app is written using Python, HTML, and JavaScript mainly. It uses D3 to read in the list of questions from our database and dynamically generate the HTML code for the questionnaire page. Upon clicking the submit button, Flask and Python unpickle the trained model and save the responses in a Pandas 1x91 DataFrame. The DataFrame is run through the model to let the model predict the user's personality trait. A results page displays the model's prediction in a Jumbotron element.
